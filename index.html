PORTFOLIO


Project 1

HandiChatCraft Mobile Application


As the leader of a five-member team for the SDGP module at IIT, we developed a mobile app for counseling Sri Lankan handicrafts using Flutter and Firebase. This offers a seamless experience for clients and counselors, featuring role-based sign-in/sign-up journeys, a home page with various handicraft categories, a community forum, a chatbot, feedback and rating options, client profiles with counselor selections, appointment scheduling, and a calendar for managing sessions. I was responsible for creating the welcome page, introduction page, sign-in/sign-up page, forgot password page, home page, and seven category pages.




Project 2

Music Website


I developed a dynamic music website using HTML, CSS, and JavaScript. The site features a sleek menu slider for easy navigation, a curated list of top artists, and various music categories. It also includes a top charts list showcasing popular tracks and a fully functional music player that allows users to listen to songs from the top charts. This project highlights my skills in front-end development and my ability to create engaging and interactive user experiences.






Project 3

Flower Boquets Course Website


I developed a flower bouquet-making course website using React.js, Node.js, and MySQL. The site includes a home page with an overview of the course, a details page outlining the three-month curriculum, a lessons page, and a tools page showcasing gifts for enrollees. It also features a feedback page with testimonials and a registration form for new users. This project highlights my expertise in full-stack development and my ability to create a functional and interactive educational platform.
